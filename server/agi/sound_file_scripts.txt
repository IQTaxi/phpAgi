TECHNICAL NOTES:
===============

1. All sound files should be in WAV format, mono channel, 8kHz sample rate
2. Keep recordings clear and at consistent volume
3. Pause slightly before and after each recording
4. For DTMF prompts, speak clearly and slowly
5. Test each file in the system before production use

RECORDING COMMANDS (if using Asterisk):
=====================================

To record directly in Asterisk console:
- originate Local/s@record-context extension s@record-context
- Use Monitor() or MixMonitor() applications
- Convert to proper format: sox input.wav output.wav rate 8000 channels 1

QUALITY CHECKLIST:
=================
☐ Clear pronunciation in Greek
☐ Appropriate speaking pace
☐ No background noise
☐ Consistent volume levels
☐ Proper file naming convention
☐ Correct audio format (WAV, 8kHz, mono)
☐ Test playback in Asterisk system
