Quick Setup Checklist:
Enable 5 Google APIs
Create API Key
Set up billing
Configure restrictions

Required Google APIs

1. Cloud Speech-to-Text API
speech.googleapis.com
Purpose: Converts customer voice input to text
Usage: Name, pickup address, destination, reservation time
Endpoint: https://speech.googleapis.com/v1/speech:recognize
2. Cloud Text-to-Speech API
texttospeech.googleapis.com
Purpose: Generates speech confirmation messages
Usage: Booking confirmations, address verification
Endpoint: https://texttospeech.googleapis.com/v1/text:synthesize
3. Maps Geocoding API (Legacy Option)
geocoding-backend.googleapis.com
Purpose: Converts addresses to GPS coordinates
Usage: Basic address geocoding
Used when: geocodingApiVersion = 1
Endpoint: https://maps.googleapis.com/maps/api/geocode/json
4. Places API (Recommended Option)
places-backend.googleapis.com
Purpose: Advanced address search with better accuracy
Usage: Enhanced geocoding and address matching
Used when: geocodingApiVersion = 2
Endpoint: https://places.googleapis.com/v1/places:searchText
5. Cloud Translation API
translate.googleapis.com
Purpose: Translates text between different languages
Usage: Date and time parsing, address translation, multi-language support
Languages: Greek to English translation and vice versa
Endpoint: https://translation.googleapis.com/language/translate/v2

Setup Instructions

Step 1: Go to Google Cloud Console
Step 2: Navigate to APIs & Services → Library and enable all 5 APIs above
Step 3: Go to APIs & Services → Credentials and create an API Key
Step 4: Restrict your API key to only the required APIs
Step 5: Add the API key to your configuration file

Support

For issues with Google APIs: Google Cloud Support

For AGI system issues: Check system logs and analytics dashboard

After Setup: Your AGI system will be able to process voice calls with speech recognition, text-to-speech, and address geocoding capabilities.
